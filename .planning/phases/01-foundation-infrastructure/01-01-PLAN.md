| Repo → git history | Source committed to GitHub; must contain no secrets |
| Go binary → PostgreSQL | DATABASE_URL credentials injected by Railway env, never in repo |

| T-01-02 | Information Disclosure | medium | WriteProblem `detail` field | mitigate | `detail` documented as developer-only; callers forbidden from passing `err.Error()` or stack traces; verified by code review of every WriteProblem call site in later phases |
